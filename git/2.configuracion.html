Instalación y configuración de Git

Para instalar Git en Windows primero te dirijes a la página git-scm.com . Vas a descargar la última versión para Windows. Elijo la versión "64-bit Git for Windows Setup" en mi caso. En el proceso de instalación puedes darle siguiente a todo.

Al final te abre una consola (si es que marcaste que te abra Git Bash) la cual le podés hacer zoom manteniendo Ctrl + ruedita del mouse o +. Lo primero que vamos a hacer es verificar tener Git instalado en su última versión con el comando:

git --version

Luego vamos a identificar nuestro nombre y correo electrónico para que luego sepan cual fue la persona que realizó cierta modificación:

git config --global user.name "Tu_nombre"

git config --global user.email "tu_email@gmail.com"

Uso el comando global porque quiero que se quede guardado en mi computadora de manera global y que no me lo vuelvan a pedir. Suponiendo que quieras tener una identidad diferente en cada proyecto puedes eliminar el comando global.

También voy a establecer el coloreado automático de la línea de comandos para facilitar la revisión en Git:

git config --global color.ui auto

Si quieres que tu rama por defecto, al inicializar un repositorio, sea siempre main en vez de master usa el comando:

git config --global init.defaultBranch main

Para comprobar tu configuración puedes escribir el comando:

git config --list
